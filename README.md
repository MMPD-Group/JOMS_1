# JOMS_1
This repository contains the machine learning and optimization framework developed for the design of advanced steel alloys. The objective of this study is to optimize elemental compositions and process temperatures to maximize mechanical properties: 0.2% Proof Stress, Tensile Strength (UTS), and Elongation %, using twenty-two elemental composition percentages and temperature as input features.

The scripts were developed using Python. The required packages include pandas, numpy, scikit-learn, xgboost, pymoo, shap, and matplotlib. The models implemented in this work primarily utilize XGBoost Regressors for predictive modeling. Multi-objective and single-objective optimization are performed using the NSGA-II evolutionary algorithm to ensure robust identification of optimal alloy compositions.

The property-specific scripts, including Proof_Stress_MOO.py, Tensile_Strength_MOO.py, and Elongation_MOO.py, correspond to scenarios where a single mechanical property is maximized simultaneously across three operational temperatures (27°C, 400°C, and 1000°C). The model is trained on the full dataset and predictions are performed to identify the optimal elemental mix that maintains high performance across a wide thermal spectrum.

The temperature-specific scripts, including MOO_27C.py, MOO_400C.py, and MOO_1000C.py, involve fixing the operational temperature and simultaneously optimizing all three mechanical properties to balance strength and ductility. Furthermore, an advanced multi-objective framework, temp_MOO.py, is implemented where temperature is treated as an independent optimization variable alongside the elemental composition, allowing the algorithm to dynamically identify the absolute optimal thermomechanical state.

The Single-Objective Optimization framework, implemented in SOO.py, follows a targeted approach to maximize one specific mechanical property at a highly specific operational temperature. This script incorporates real-time generation callbacks to track the algorithm's progress and isolate absolute maximum theoretical values for highly specialized applications.

To evaluate the physical feasibility of the generated compositions, interpretation and validation scripts are included. Shap_code.py includes full feature interpretation using SHAP to quantify the contribution of individual elemental additions toward mechanical performance. Concurrently, knn_cleaned.py uses a k-Nearest Neighbors algorithm to compare the algorithmically generated optimal compositions against the closest real-world experimental samples in the training dataset. The data used for all optimization cases is provided in an Excel file included in this repository.
